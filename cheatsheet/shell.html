Shell reminder points Shell cheatsheet
  • cd in Windows / pwd Mac/OSX is used to print working directory in windows
  • dir in Windows or ls in Mac/OSX list directory contents
  • cd dir in Windows or in Mac/OSX is for changing to a subdirectory
  • cd .. in both systems to go up a directory
  • mkdir dir to create a directory
  • rmdir dir in both to delete a directory
Command in Windows Command in Mac/OSX Purpose
cd pwd print working directory
dir ls list directory contents
cd dir cd dir for changing to a subdirectory
cd .. cd .. go one directory up
mkdir directoryname mkdir directoryname create a directory
rmdir an existing directory rmdir an existing directory remove an existing direcotry
